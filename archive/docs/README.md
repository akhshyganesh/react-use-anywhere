## 🛠️ Development

Want to contribute to docs?

### Run docs locally:

```bash
cd docs
npm install
npm run dev
```

Visit: `http://localhost:5173/react-use-anywhere/`

### Build docs:

```bash
cd docs
npm run build
```

### Deploy docs:

```bash
npm run docs:deploy
```
