# Changelog

## [1.1.0] - Current

### Added

- Comprehensive documentation site with VitePress
- Live demo application showcasing all features
- Complete API reference documentation
- Step-by-step guides and tutorials
- Real-world usage examples
- Troubleshooting guide
- GitHub Pages deployment setup

### Improved

- Enhanced type safety across the library
- Better error handling and debugging capabilities
- Improved service layer architecture
- More robust hook injection system

## [1.0.0] - Initial Release

### Added

- Core hook injection system with `HookInjectionProvider`
- `useHookService` hook for accessing injected services
- `createHookService` utility for creating service instances
- TypeScript support with full type safety
- Router-agnostic design
- Comprehensive test suite
- Basic README and setup instructions

### Features

- Service injection and dependency management
- Type-safe hook usage patterns
- Flexible service layer architecture
- React 18+ compatibility
- Zero runtime dependencies (except peer dependencies)

## Migration Guide

### From 1.0.x to 1.1.x

No breaking changes. All existing code will continue to work as before. The 1.1.x release focuses on documentation, examples, and developer experience improvements.
