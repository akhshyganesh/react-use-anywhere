# Examples

This directory contains working examples of how to use the React Hook Injection Pattern library in different scenarios.

## Available Examples

### 1. Basic Navigation (`basic-navigation/`)
Simple navigation service usage with React Router DOM.

### 2. Authentication Flow (`auth-flow/`)
Complete authentication flow with protected routes and automatic redirection.

### 3. Multiple Services (`multi-service/`)
Demonstrates using multiple services with different hooks injected.

### 4. TypeScript (`typescript/`)
Full TypeScript implementation with strict type checking.

### 5. Testing (`testing/`)
Comprehensive testing examples for services and components.

## Running Examples

Each example is a complete, runnable React application:

```bash
cd examples/basic-navigation
npm install
npm start
```

## Example Structure

Each example follows this structure:
```
example-name/
├── src/
│   ├── components/
│   ├── services/
│   ├── hooks/
│   └── App.tsx
├── package.json
└── README.md
```
